# JavaScript 初学者项目模板

这是一个简单的 JavaScript 项目，适合初学者学习和练习。

## 项目结构

```
.
├── index.html      # 主 HTML 文件
├── style.css       # 样式文件
├── app.js          # JavaScript 代码
├── package.json    # 项目配置文件
└── README.md       # 说明文档
```

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或者使用 VS Code 的 Live Server 插件

## 项目包含的功能

### 1. 计数器
- 点击 "+" 按钮增加计数
- 点击 "-" 按钮减少计数
- 点击 "重置" 按钮将计数归零

### 2. 打招呼
- 在输入框中输入你的名字
- 点击 "打招呼" 按钮或按 Enter 键
- 页面会显示个性化的问候语

## 学习要点

这个项目涵盖了以下 JavaScript 基础知识：

1. **DOM 操作**
   - `document.getElementById()` - 获取页面元素
   - `element.textContent` - 修改元素文本内容

2. **事件处理**
   - `addEventListener()` - 添加事件监听器
   - 点击事件 (`click`)
   - 键盘事件 (`keypress`)

3. **变量和函数**
   - 使用 `let` 声明变量
   - 定义和调用函数

4. **条件判断**
   - `if...else` 语句

5. **Console 调试**
   - `console.log()` - 在控制台输出信息

## 下一步学习建议

可以尝试添加以下功能来练习：

- [ ] 为计数器添加最大值和最小值限制
- [ ] 添加更多的交互元素（如待办事项列表）
- [ ] 尝试使用 `localStorage` 保存数据
- [ ] 添加更多的 CSS 动画效果
- [ ] 创建一个简单的计算器

## 调试技巧

- 按 `F12` 打开浏览器开发者工具
- 在 Console 标签页可以看到 `console.log()` 的输出
- 使用 Elements 标签页查看和修改 HTML/CSS
- 在 Sources 标签页可以设置断点调试 JavaScript
