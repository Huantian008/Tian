Web前端开发技术

05月20日11:06 前端学习网

Web前端开发技术包含三部分：HTML、CSS、JS。 HTML用于设定网页结构，也就是网页包括哪些内容。 CSS用于设定网页样式，主要包括页面元素的样式和网页布局。 JS是一种客户端脚本语言，它能够设定网页行为，用于完成网页一些动态效果、与用户的交互效果。

名词解释

名词 解释
HTML HyperText Markup Language超文本标记语言
CSS Cascading Style Sheets层叠样式表
JS JavaScript语言
